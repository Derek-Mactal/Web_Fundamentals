First appearing in 1991, Python is a general-purpose, high-level, interpreted programming launguage whose design focus emphisizes code readability. Whether for processing/analyzing data to powering many well-known applications, Python is not only a great language for new programmer but also powerful enough for seasoned developers building whatever they can think of.
